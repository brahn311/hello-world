# Aliases for git

#### Use this comand at gitbash and and
`git config --global alias.[referencia] "[comando]"`
```
git config --global alias.s "status"
git config --global alias.aa "add ."
git config --global alias.cm "commit -m"
```
#### Or add at `.gitconfig`
```
[alias]
	s = status
	aa = add .
	cm = commit -m
```
